# Ocni

Ocni is a project consisting of 2 web apps. One helps with the management of an eye clinic while the other is basically an e-shop for glasses.

## Technologies familiarised with

* HTML, CSS, JS, JQUERY
* PHP, MySQL
* Composer
* Laravel - Eloquent
* PHPMailer
* Bramus\Router
* flatpickr
* MVC Architecture
